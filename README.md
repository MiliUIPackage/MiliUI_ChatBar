# MiliUI ChatBar

A simple and lightweight chat channel quick-access bar addon for World of Warcraft.

## Features

- 🖱️ **Right-click Menu** - Right-click on buttons to open a context menu for additional options
- ✋ **Drag to Reposition** - Enable Edit Mode to drag and reposition the chat bar anywhere on your screen
- 🌐 **Multi-language Support** - Supports English, Traditional Chinese (繁體中文), Simplified Chinese (简体中文), and Korean (한국어)

## Installation

1. Download the addon
2. Extract the `MiliUI_ChatBar` folder to your `World of Warcraft/_retail_/Interface/AddOns/` directory
3. Restart World of Warcraft or type `/reload` in-game

## Usage

- **Left-click** on a channel button to switch to that chat channel
- **Right-click** on a background to open the context menu
- Use **Edit Mode** (Esc > HUD Edit Mode) to drag and reposition the chat bar

## Contributing

Contributions are welcome! Feel free to submit pull requests to help improve this addon.

# MiliUI 快捷聊天列

一款簡單輕便的魔獸世界聊天頻道快捷按鈕列插件。

## 功能特色

- 🖱️ **右鍵選單** - 右鍵點擊按鈕可開啟選單以使用更多功能
- ✋ **拖曳定位** - 在編輯模式中拖曳聊天列以調整位置

1. 下載插件
2. 將 `MiliUI_ChatBar` 資料夾解壓縮至 `World of Warcraft/_retail_/Interface/AddOns/` 目錄
3. 重新啟動魔獸世界或在遊戲中輸入 `/reload`

- **左鍵點擊** 頻道按鈕可切換至該聊天頻道
- **右鍵點擊** 按鈕列背景處可開啟選單

# MiliUI 채팅바

월드 오브 워크래프트를 위한 간단하고 가벼운 채팅 채널 빠른 접근 바 애드온입니다.

## 기능

- 🖱️ **우클릭 메뉴** - 버튼을 우클릭하여 추가 옵션이 있는 컨텍스트 메뉴 열기
- ✋ **드래그하여 위치 변경** - 편집 모드를 활성화하여 채팅 바를 화면 어디든 드래그하여 재배치
- 🌐 **다국어 지원** - 영어, 繁體中文, 简体中文, 한국어 지원

## 설치 방법

1. 애드온 다운로드
2. `MiliUI_ChatBar` 폴더를 `World of Warcraft/_retail_/Interface/AddOns/` 디렉토리에 압축 해제
3. 월드 오브 워크래프트를 재시작하거나 게임 내에서 `/reload` 입력

## 사용 방법

- 채널 버튼을 **좌클릭**하여 해당 채팅 채널로 전환
- 버튼을 **우클릭**하여 버튼바를 우클릭하여 컨텍스트 메뉴 열기
- **편집 모드** (Esc > HUD 편집 모드)를 사용하여 버튼바를 드래그하여 재배치
